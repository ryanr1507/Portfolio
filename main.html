Holder Profile Builder

NX has a built in database for holder profiles. I created a UI that requires the user to select the holder and the front face of the selected holder. The program collects all the edges starting from the front face of the holder and calculates the length of each section, diameter, and taper. The previous calculations along with a library reference, sequence number, and other required data are formated to create the holder profile. These custom profiles can be stored in the database so they can be imported to an active session. The UI includes a feature which allows the user to move the holder to the absolute coordinate system and align it with the X axis.

Job Scheduler

The purpose of this program is to quickly give an estimated completion date for any given job. The user can change any settings on the main page including machines, operations, and work hours. The 'Start Program' button will prompt the user with a form and begin the scheduling process. An operation, machine, and operation time (time it takes to finish an operation) can be added in the top portion of the form. Once all operations are added for the job, the user can add a start date for each group and number of parts. The program takes the first start date and will start scheduling based off operation times, work hours for the weekdays/weekends, and what machines are available so theres no overlap. This program also includes a time delay function which allows the user to select any cells and delay the time.
